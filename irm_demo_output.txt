📚 ABOUT LATTICE STRUCTURES:
Lattice-ladder structures are particularly valuable for:
  • Real-time parameter modulation (frequency, Q, gain)
  • Avoiding transients/glitches when updating coefficients
  • Improved numerical stability
  • Coefficient sensitivity reduction

As noted by Robert Bristow-Johnson (RBJ):
'If you want to vary parameters continuously while passing signal,
you are not supposed to use biquads (because updating coefficients
causes glitches) and you should use state-variable filters instead.'

Lattice structures provide an alternative approach for smooth parameter updates.

✅ Coefficient Conversion Formulas:
  • k2 = a2
  • k1 = a1 / (a2 + 1)
  • v2 = b2
  • v1 = b1 - a1 * b2
  • v0 = b0 - (a1/(a2+1))*b1 + ((a1²/(a2+1)) - a2)*b2

✅ Transfer Function:
  H(z) = (v0 + v1*(k1+z⁻¹) + v2*(k2+k1*(k2+1)*z⁻¹+z⁻²)) / (1 + k1*(k2+1)*z⁻¹ + k2*z⁻²)

✅ Implementation: Perfect equivalence to direct form biquad

This implementation enables real-time parameter modulation without
the glitches that occur when updating biquad coefficients directly.
